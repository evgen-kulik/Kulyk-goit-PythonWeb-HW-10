Реалізовано:
1. Можливість реєстрації на сайті та вхід на сайт для зареєстрованого користувача.
2. Можливість додавання нового автора для зареєстрованого користувача.
3. Можливість додавання нової цитати із зазначенням автора для зареєстрованого користувача.
4. Робота ведеться з базою Postgres.
5. Можна зайти на сторінку кожного автора без автентифікації користувача.
6. Усі цитати доступні для перегляду без автентифікації користувача.

Додаткова частина:
1. Реалізовано пошук цитат за тегами. При натисканні на тег, виводиться список цитат з цим тегом.
2. Реалізовано пагінацію. Це кнопки next та previous.
